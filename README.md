# Java 텍스트 기반 TRPG 미니 프로젝트
간단한 TRPG 게임 기능 구현(레벨업,스킬,상태이상), 저장/불러오기 기능을 구현한 텍스트 기반 자바 미니 프로젝트 입니다.
## 주요기능
- 턴제 전투 시스템: 플레이어와 몬스터가 전투 시 주고받으며 턴이 진행됩니다.
- 다양한 몬스터: Slime, Goblin, Wolf 3가지의 몬스터 객체 구현
- 상태창과 경험치: 플레이어의 현재 상태를 확인 할 수 있고, 경험치를 획득해 레벨업을 할 수 있습니다.
- 스킬과 상태이상: 플레이어가 일정 레벨 달성 시 스킬을 획득하며, 스킬을 사용하여 확률적으로 몬스터에게 상태이상을 적용시킬 수 있습니다.
- 저장 기능: 진행된 내용을 저장하고, 불러오기 기능을 통해 저장된 내용을 불러올 수 있습니다.

## 기능 요소
1. exception 처리를 사용한 이유.

시스템의 안정성을 확보하기 위해서 예외처리를 진행했습니다.
스킬을 사용 할 때 MP가 부족같은 예외적인 상황이 발생했을 때, 프로그램의 비정상적인 종료를 방지하기 위해 Exception 을 사용했습니다.

2. Extends와 Interface를 구현한 이유.

플레이어와 몬스터의 공통속성과 기능을 Entity에서 정의하고,이를 상속받아 중복코드를 최소화하고,효율적으로 관리하기 위해 Extends를 사용했습니다.
전투를 진행 할 수 있는 객체의 규격을 명시하고, 전투로직의 에러를 최소화하여 전투로직을 유연하게 만들기 위해서 Interface를 사용했습니다.

3. 컬렉션 프레임워크 사용 이유.

중복 코드를 줄이고 게임 데이터의 효율적인 관리를 위해서 컬렉션 프레임워크를 사용했습니다.

## 추가기능 예상
1. 다양성 추가 - 몬스터,스킬,상태이상의 종류를 추가하여, 더 다양한 플레이가 가능하게 할 수 있습니다.
2. 아이템 추가 - 아이템의 기능과 종류를 구현하여, 플레이어가 인벤토리 기능을 사용하고 아이템을 활용 할 수 있습니다.
3. 상점 시스템 추가 - 아이템의 구매/판매 기능을 추가하여, gold의 획득/소모를 가능하게 할 수 있습니다.
4. 퀘스크 시스템 추가 - 퀘스트 기능을 추가하여, 일반적인 전투승리 외에도 경험치/골드를 추가 획득 힐 수 있습니다.
